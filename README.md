# Church Leader Training Hub – Website Project

## Overview
The Church Leader Training Hub is a multi-page website designed to support pastors, ministry leaders, and volunteers in building healthy, sustainable ministry teams. The purpose of this site is to provide practical training resources and tools that can be easily applied in real church environments.

This project was created as part of a web design course and demonstrates the application of design principles, layout structure, color theory, and responsive web design.

---

## Website Structure
The website includes the following main pages:

- **Home** – Introduces the purpose of the site and provides an overview of available resources  
- **Training Resources** – Contains short, focused content on leadership development and team building  
- **Practical Tools** – Provides templates and systems for organizing and supporting ministry teams  
- **About** – Explains the leadership philosophy and purpose behind the website  
- **Contact** – Allows users to reach out for additional information or support  

Each page includes relevant images and text content to support usability and visual consistency.

---

## Design Features
- Custom color scheme using warm and neutral tones  
- Consistent typography and layout across all pages  
- Use of real ministry-related images  
- Application of design principles:
  - Alignment
  - Proximity
  - Repetition
  - Contrast  
- Responsive layout for different screen sizes

## How to View the Website
1. Download and unzip the project folder  
2. Open the file **index.html** in a web browser  
3. Navigate through the site using the navigation bar
